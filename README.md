# Car Price Prediction with Machine Learning

This project is a Machine Learning application that predicts car prices using various car-related features such as brand value, horsepower, mileage, fuel type, engine specifications, and other important attributes. The main objective of this project is to understand the workflow of a regression-based Machine Learning problem, including data preprocessing, feature engineering, model training, testing, and performance evaluation.

The project uses Python libraries such as Pandas, NumPy, Matplotlib, and Scikit-learn for data handling, visualization, and model building. A Linear Regression model is implemented to analyze the relationship between input features and car prices. The dataset is divided into training and testing sets to evaluate the prediction capability of the model on unseen data.

Model performance is measured using evaluation metrics such as R Squared Error and Mean Absolute Error (MAE). Prediction graphs are also generated and saved to visualize the relationship between actual and predicted car prices for both training and testing datasets.

This project helps beginners understand regression algorithms, data preprocessing techniques, model evaluation methods, and real-world applications of Machine Learning in price prediction systems.
